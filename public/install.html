Installation Guide

Follow these steps to install PDF Reader on your computer

1

Download the Installer

Click the "Update Now" button on the previous page to download the latest PDF Reader installer. The file will be saved to your Downloads folder.

[Screenshot: Download button highlighted]
2

Locate the Downloaded File

Open your Downloads folder and find the installer file. It will be named something like:

pdf-reader-installer.exe (Windows)
pdf-reader-installer.dmg (macOS)
pdf-reader-installer.deb (Linux)
3

Run the Installer

Double-click the installer file to begin the installation process. You may see a security warning - this is normal. Click "Yes" or "Allow" to continue.

[Screenshot: Windows security dialog]
4

Follow Installation Wizard

The installation wizard will guide you through the process:

  • Accept the license agreement
  • Choose installation location (default is recommended)
  • Select components to install
  • Click "Install" to begin
[Screenshot: Installation wizard welcome screen]
5

Wait for Installation to Complete

The installer will extract and install all necessary files. This usually takes 1-3 minutes. A progress bar will show the installation status.

[Screenshot: Installation progress bar]
6

Complete Installation

When installation is complete, you'll see a confirmation message. Click "Finish" to exit the installer. You may be asked to restart your computer.

[Screenshot: Installation complete screen]
7

Launch PDF Reader

After installation, you can launch PDF Reader by:

  • Windows: Start Menu → PDF Reader
  • macOS: Applications folder → PDF Reader
  • Linux: Applications menu → PDF Reader

You can also set PDF Reader as your default PDF viewer in your system settings.

System Requirements

  • Operating System: Windows 10/11, macOS 10.15+, or Linux (Ubuntu 20.04+)
  • Processor: 1 GHz or faster
  • RAM: 2 GB minimum (4 GB recommended)
  • Disk Space: 500 MB available space
  • Display: 1024x768 resolution or higher
  • Internet: Required for initial activation and updates

Troubleshooting

Installation fails or shows error messages:

  • Make sure you have administrator privileges
  • Disable antivirus temporarily during installation
  • Ensure you have enough disk space
  • Try downloading the installer again

PDF files won't open after installation:

  • Right-click a PDF file → Open with → Choose PDF Reader
  • Set PDF Reader as default in system settings
  • Restart your computer

Still having issues?

Contact our support team at support@pdfreader.com or visit our help center.

Important Notes

  • Always download PDF Reader from official sources only
  • Keep your software up to date for the latest security patches
  • If updating an existing installation, your settings and preferences will be preserved
  • During installation, you may be asked to close any running PDF files
  • Installation requires administrator rights on Windows and macOS
